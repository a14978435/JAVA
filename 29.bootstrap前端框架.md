# bootstrap前端框架

## 通用颜色

| Class      | 描述                                         |
| :--------- | :------------------------------------------- |
| `.active`  | 鼠标悬停在行或单元格上时所设置的颜色【深色】 |
| `.success` | 标识成功或积极的动作【绿色】                 |
| `.info`    | 标识普通的提示信息或动作【蓝色】             |
| `.warning` | 标识警告或需要用户注意【黄色】               |
| `.danger`  | 标识危险或潜在的带来负面影响的动作【红色】   |

## 按钮Button

* btn 按钮样式
* btn-success 成功按钮(绿色)
* btn-info 信息按钮(蓝色)
* btn-danger危险按钮(红色)
* btn-block 沾满一行

## FORM表单

* form-horizontal【表单项间隔】

## 输入框Input

* form-control 一行输入框
* disabled 禁止输入
* has-success 绿边框
* has-warning 黄边框
* has-error 红边框
* input- sm，lg 小，大输入框

## 列表 ul,dl...

* pagination 标签页 (li横排块)
* list-unstyleed 无样式ul
* list-inline 横排列表
* dl-horizontal 水平排列的描述
